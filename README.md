### 项目使用

---
+ 服务端
    - node版本 6.x+
    - npm 3.x+
```
    安装 npm i 
    运行 npm run dev
```

+ 客户端
    - vue版本 2.x+
    - npm 3.x+
```
    安装 npm i 
    运行 npm run dev
```
